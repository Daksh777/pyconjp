Robust System Design in Python — The Benefits of Immutability

MK
Masaya Kato
Backend Engineer
slideshow Session slides speakerdeck.com/masaya00/robust-system-design-immutability open_in_new

Abstract

This talk explores why immutable data structures matter in Python, and how you can lean on them to design more robust systems.

Most Python developers reach for mutable data structures like lists and dictionaries every day — and most have, at some point, been bitten by the unexpected side effects and bugs they cause. Mutable objects become especially troublesome when data is shared across modules, or when concurrency enters the picture.

Drawing on lessons learned building real-world systems, this talk covers:

  • Common problem patterns that mutable code tends to cause
  • Immutable design techniques using frozen dataclasses and properties
  • Practical use cases in web applications (databases and beyond)

Using concrete examples pulled straight from production code, the talk introduces practical techniques you can put to work right away. By pairing Python's type hints with immutable design, you'll learn how to build safer, more maintainable codebases.

It is aimed at developers building systems in Python, and at teams looking to raise the quality bar of their code together.

Motivation

I used to write code without giving much thought to the risks that come with objects being mutable.

Then, on one project, I committed to a single simple rule — "once an object is created, never change it." Across every part of development, from implementation to review to debugging, the nagging question of "has this value been changed somewhere?" simply went away. That freed me to pour my energy into the parts that genuinely deserved deep thought.

Through that experience, I came to believe that the predictability and mental simplicity immutable design brings are a powerful ally for developers — regardless of a project's size or type.

I chose this topic to share the immutable design practices I rely on in my own work, in the hope that it helps you build even slightly more robust web applications.
