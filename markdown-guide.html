Markdown Style Guide

Every page on this site can be authored as a plain .md file. This is the living reference for how each Markdown element is rendered into the PyCon JP 2026 design system — write Markdown, get this.

👋 How to read this page. The left column shows the Markdown syntax you type; the styled result is what visitors see. Write your content in content/*.md, and the site converts it using exactly the styles below.

#Front matter

Every .md file starts with a YAML front-matter block, fenced by ---. It sets the page title, date, and other metadata the site uses to build the page, its URL, and its listing — it is not shown as body content. 

yaml---
title: "Call for Proposals is open"
date: 2026-03-01
category: news
lang: en
tags: [cfp, speakers]
draft: false
---

#Headings

Use one # H1 per page for the title, then nest ## through ###### for structure. Headings also generate the sidebar anchors.

H1 — The page title

H2 — A major section

H3 — A subsection

H4 — A minor heading

H5 — A small label
H6 — An eyebrow / overline

#Text & emphasis

This is a normal paragraph. You can make text bold with **bold**, italic with *italic*, or both. Strike things out like this old date with ~~strikethrough~~, and highlight key phrases.

Inline extras: inline code, a keyboard hint like Ctrl + K, an PEP abbreviation, H2O subscript, 21st superscript, and small print.

Python is a great language for beginners and experts alike — and PyCon JP 2026 is the place to level up, whether you write web apps, crunch data, or teach.

#Links & buttons

Standard links use [label](url): read the conference overview, review the Code of Conduct, or open the Python website ↗ in a new tab. Auto-links work too: info@pycon.jp.

Auto-links (a bare URL in angle brackets) render clickable too: https://2026.pycon.jp.

Buttons & CTAs

Promote a link to a call-to-action button by adding a class. The primary and ghost styles share the same size:

Buy Tickets → Become a Sponsor

A compact variant for inline or secondary actions:

Learn more → View venue

Or full-width, for a strong page-level call to action:

Register for PyCon JP 2026 →

#Definition lists

Pair a term with its description — ideal for glossaries and FAQ-style key/value content:

Keynote
An invited, plenary talk that opens each conference day.
Sprint
A hands-on day where attendees contribute to open-source projects together.
LT
Lightning Talk — a fast, five-minute talk anyone can give.
CFP
Call for Proposals — the window during which you can submit a talk.

#Blockquotes

“Come for the language, stay for the community.”

— A frequent PyCon JP attendee

Blockquotes can nest, which is handy for quoting a reply:

Will there be childcare at the venue this year?

Yes — free childcare is available on all three main-conference days. Register when you buy your ticket.

#Code

Use single backticks for inline code — great for names like venv, pip install, or __init__.py. For blocks, fence with triple backticks and a language tag: 

python# A tiny greeting for every attendee
def welcome(name: str) -> str:
    """Return a friendly hello."""
    return f"Welcome to PyCon JP 2026, {name}!"


for attendee in ["Sana", "Ken", "Priya"]:
    print(welcome(attendee))

Shell commands render the same way:

bash# Set up your environment before the sprint
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

A diff block highlights additions and removals — great for changelogs:

diff  schedule:
-   day1: "Talks only"
+   day1: "Keynote + Talks"
+   day3: "Sprints"

#Callouts & admonitions

Use blockquote-style admonitions (> [!NOTE], > [!TIP], etc.) to draw attention:

Note

All main-conference talks are recorded and published on our YouTube channel a few weeks after the event.

Tip

Buy your ticket early — early-bird pricing ends when the first batch sells out.

Warning

The venue has limited on-site parking. Public transport to ICC Hiroshima is strongly recommended.

Important

All attendees, speakers, and sponsors must follow the Code of Conduct.

#Collapsible sections & FAQ

Wrap content in <details> and <summary> to create click-to-expand sections — perfect for FAQs, long schedules, or optional detail. Click a question to open it:

Do I need to know Japanese to attend?

No. Talks are given in both Japanese and English, and a live translation app is provided for most sessions. Signage and volunteers can help in English throughout the venue.

Are tickets refundable?

Tickets are transferable up to one week before the event. Refunds are handled case by case — email info@pycon.jp with your order number.

What's included with a conference ticket?
  • Access to all talks and the keynote on Days 1–2
  • Lunch and coffee on both conference days
  • The sprint day on Day 3
  • Conference swag and a name badge

#Rules, line breaks & escaping

A horizontal rule (---) renders as the site's dotted divider to break up long content:

Hard line breaks

End a line with two spaces (or a \) to force a break without starting a new paragraph:

PyCon JP 2026
ICC, Hiroshima
August 21–23

Escaping & literal characters

Backslash-escape a character to show it literally: \*not italic\* renders as *not italic*, and \# shows a literal #. Use &nbsp; to keep dates like Aug 21–23 from wrapping, and &mdash; for an em dash — like this.

#Footnotes

Add a reference with [^1] in your text and define it anywhere in the file. The site collects them at the bottom of the page with links back to each reference. PyCon JP began in 2011[1] and now draws over a thousand attendees[2] each year.
